# OWASP

**Open Worldwide Application Security Project** — a nonprofit foundation that works to improve software security through community-led open source projects, documentation, and tooling.

## Notable Resources

- **Cheat Sheet Series**: Concise, actionable security guidance for developers. Covers 100+ topics including Docker, IaC, Kubernetes, authentication, injection prevention, etc.
- **OWASP Top 10**: Most critical web application security risks
- **OWASP Docker Top 10**: https://github.com/OWASP/Docker-Security
